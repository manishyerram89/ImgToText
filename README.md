# Tools Used

This project utilizes the following tools and libraries:

- **PIL (Pillow)** – For image loading and preprocessing.
- **glob** – For reading file paths in batches.
- **Google Colab Drive** – For mounting and accessing external storage.
- **tqdm** – For progress visualization during OCR operations.
- **EasyOCR** – Core OCR engine used for text extraction from images.
- **Matplotlib** – For visualizing images and OCR outputs.
- **NumPy** – For numerical operations and array manipulation.
- **Pandas** – For organizing extracted data into structured formats.
